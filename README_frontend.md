# ATOM ДОБРО

Карта добрых дел атомных городов - веб-приложение для просмотра и добавления волонтерских организаций и квестов на интерактивной карте.

## Описание

Приложение позволяет пользователям:
- Просматривать волонтерские организации на карте атомных городов
- Добавлять новые организации и квесты
- Фильтровать организации по городам, типам помощи и направлениям
- Просматривать профиль с достижениями, статистикой и историей участия
- Участвовать в квестах и получать опыт

## Технологии

- React 19
- TypeScript
- Vite
- Redux Toolkit + RTK Query
- React Router (file-based routing)
- Leaflet (карты)
- Tailwind CSS
- React Hook Form + Zod
- Redux Persist

## Установка

1. Установите зависимости:
```bash
pnpm install
```

2. Запустите dev-сервер:
```bash
pnpm dev
```

3. Откройте браузер по адресу `http://localhost:5173`

## Сборка

Для production сборки:
```bash
pnpm build
```

Для preview сборки:
```bash
pnpm preview
```

## Структура проекта

```
src/
  components/     - React компоненты (формы, карта, профиль, UI)
  pages/         - Страницы приложения
  store/         - Redux store и сервисы
  hooks/         - Кастомные хуки
  contexts/      - React контексты (User, Notification)
  utils/         - Утилиты
  types/         - TypeScript типы
  constants/     - Константы
```

## Основные страницы

- `/` - Главная страница
- `/map` - Интерактивная карта с организациями
- `/add-organization` - Форма добавления организации
- `/profile` - Профиль пользователя
- `/login` - Вход
- `/registartion` - Регистрация
